

### 💫 About Me
I am a multi-disciplinary tech enthusiast exploring the intersections of product design, engineering, and data. Rather than sticking to just one lane, I thrive on understanding the full software lifecycle from how a user experiences an interface to how data flows under the hood and how reliably the system performs.

-  ⚙️ **Development & QA:** Writing clean backend/frontend code and building thorough test mentalities to ensure bug-free deployments.
- 🎨 **UI/UX & Frontend:** Ideating intuitive interfaces and translating them into responsive layouts.
- 📊 **Data Science & Operations:** Querying data, uncovering insights, and playing with machine learning architectures.
